**Introduction**<br />
Segment Routing(SR) is an emerging network technology which aiming to increase the overall network performance through different paths between source and destination. Several routing algorithms existed for this propose, this project provide the ability to validate and evaluate these algorithms, by providing a GUI to analysis network graphs.
